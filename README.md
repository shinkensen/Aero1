# Aero1
A basic RC plane utilizing ESP32, Dual 1020 motors, a 500mah battery, and a pcb to coordinate everything

How ____?
- Controls:
  Elevator - Servo at the back (supplied by me)
  Yaw- Thrust control on both motors
  Controls via website
- Power:
  1s 25c Lipo Battery 3.7v -> Brushed 1020s
